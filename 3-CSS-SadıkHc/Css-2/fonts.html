bizim sayfamızda font kendi bilgisayarımızda tanımlı olan fonta göre gösteriliyor. örneğin bir başkasının bilgisayarında font özellikleri farklıysa sitemizde girdiğinde o font ile yazıları görecektir. Bunu bir standarda bağlamamız lazım. Bizimkinde de onunkinde de aynı fontta açılmasını sağlamamız lazım. css font şeklinde arama yap. 5 adet font ailesi var. Bunlar Serif , sans-serif , monospace, cursive ve fantast bu ailelerin içerisinde farklı font isimleri yer alıyor. örneğin : .p1 { font-family: "Times New Roman", Times, serif; } Burada dediğimiz şey serif ailesindeki times new roman fontunu uygula. .p2 { font-family: Arial, Helvetica, sans-serif; } burada da sans-serif ailesindeki öncelikle arial fontunu uygula. eğer arial desteklenmiyorsa helvatica kullan. demek istiyoruz. Eğer helvatica da desteklenmiyorsa sans-serif ailesinden bir tanesini seç diyoruz.
